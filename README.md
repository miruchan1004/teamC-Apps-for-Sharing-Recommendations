# teamC-Apps-for-Sharing-Recommendations
アニマルをウォッチして癒されよう

## プロダクトゴール
アプリをワンクリックするだけでみんなが癒される動画が見れる

## 実装内容
- 起動時はログイン画面を表示→ログイン後ホーム画面
- 画面下部中央に「ホーム」「検索」「マイページ」のナビゲーションを配置
- ホーム画面はかわいい動物写真を1枚ずつ縦スクロールで表示
- 検索画面では中央上に「検索欄」、その下に「検索履歴」を表示
- マイページでは中央に「アイコン」、その下に「閲覧履歴」「いいね欄」「プロフィール」を表示

## 写真の差し替え
- ホーム画面の写真は `index.html` の `photoItems` 配列で管理しています
- 使いたい画像のURLやローカルファイルに差し替えると、そのままホーム画面に表示できます

## 起動方法
`index.html` をブラウザで開いて確認してください。
